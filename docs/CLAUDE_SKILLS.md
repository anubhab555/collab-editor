# Java Backend Skill Notes

This file is a compact reminder of the backend skills demonstrated by the current Java version.

## Backend Skills

* Java 21 backend development
* Spring Boot REST APIs
* Spring Security JWT authentication
* BCrypt password hashing
* Service-layer document authorization
* Spring WebSocket realtime routing
* Redis Pub/Sub fanout
* Spring Data MongoDB persistence
* Gradle backend build
* Docker Compose packaging
* Health, readiness, and metrics endpoints

## System Design Skills

* Room-based realtime partitioning
* Stateful WebSocket scaling problem
* Redis as fanout, not durable storage
* MongoDB as durable document store
* CRDT responsibility split between frontend Yjs and Java backend routing
* Version history through stored snapshots
* Live restore as a room-wide event
